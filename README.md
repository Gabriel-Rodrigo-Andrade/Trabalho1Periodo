 **System MyHolerite**

 ## Sobre o Sistema 

   O sistema MyHoletite serve os funcionários de uma determinada empresa terem acesso ao seu própio 
 holerite apenas fazendo o login OU se cadastrando com suas informações pessoais. 
   Após efetuar o login no sistema o funcionário deve escolher qual o tipo de contratação que ele se encaixa,
 sendo elas: Efetivo, Temporário e Estagiário.
   Depois de escolher uma destas três opções o sistema solicita algumas informações antes de retornar o holerite,
  apenas aos estagiários que não é solicitado nenhuma informação pois seu salário é fixo.
